## рЯЪА Features

- рЯФЃ **Real-time prediction** вАФ enter patient data and get instant HIGH/LOW risk output with probability scores
- рЯІђ **Auto-engineered features** вАФ 4 derived features (`hr_reserve`, `pressure_ratio`, `bmi_proxy`, `risk_score`) calculated on the fly
- рЯЪ© **Live Risk Flags** вАФ highlights concerning clinical values instantly (high cholesterol, hypertension, exercise angina, etc.)
- рЯУК **Model Info dashboard** вАФ accuracy metrics, model comparison table, feature importance chart
- рЯЫ°пЄП **Demo mode** вАФ runs a rule-based fallback if `.pkl` files are missing, so the UI is fully explorable without a trained model
- рЯО® **Dark themed UI** вАФ custom crimson/dark CSS styling throughout

## рЯ©Ї Input Features

| Feature | Type | Description |
|---|---|---|
| `age` | int | Age in years |
| `sex` | 0/1 | 0 = Female, 1 = Male |
| `cp` | 0вАУ3 | Chest pain type |
| `trestbps` | int | Resting blood pressure (mmHg) |
| `chol` | int | Serum cholesterol (mg/dl) |
| `fbs` | 0/1 | Fasting blood sugar > 120 mg/dl |
| `restecg` | 0вАУ2 | Resting ECG results |
| `thalach` | int | Maximum heart rate achieved |
| `exang` | 0/1 | Exercise induced angina |
| `oldpeak` | float | ST depression induced by exercise |
| `slope` | 0вАУ2 | Slope of peak exercise ST segment |
| `ca` | 0вАУ3 | Number of major vessels (fluoroscopy) |
| `thal` | 0вАУ3 | Thalassemia type |

**Auto-Engineered Features (calculated in app):**

| Feature | Formula | Meaning |
|---|---|---|
| `hr_reserve` | `thalach вИТ (220 вИТ age)` | Heart-rate reserve estimate |
| `pressure_ratio` | `trestbps / thalach` | BP-to-heart-rate ratio |
| `bmi_proxy` | `chol / age` | Cholesterol-age ratio |
| `risk_score` | `(fbs + exang + (cp > 0)) / 3` | Simple binary risk index |
